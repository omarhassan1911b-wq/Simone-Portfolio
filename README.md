# Simone - Personal Portfolio 

Welcome to my first personal portfolio project! I built this website while learning the basics of **Front-End Development**. It's a clean, modern, and fully responsive design.

## Why I built this?
This project was a great way for me to practice:
* Structuring web pages with **HTML5**.
* Styling and layout with **CSS3** (Flexbox & Responsive Design).
* Creating animations like the "Typing Effect" in the hero section.
* Organizing project files and assets.

## Technologies Used
* **HTML5** & **CSS3**
* **Font Awesome Icons**
* **Google Fonts**

##  Key Features
* **Fully Responsive**: Works on mobile, tablets, and desktops.
* **Typing Animation**: A dynamic effect for my job titles.
* **Smooth Navigation**: Users can jump between sections easily.
